# using_ssl
Структура каталогов проекта:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
root
├── common          - общий код, используемый в подпректах
├── other           - посторонний код пока не имеющий собственного имени (временная директория)
├── ssl_client      - клиент, использующий протокол SSL
├── ssl_server      - сервер, использующий протокол SSL
└── stuff           - вспомогательное барахло для работы с SSL (сертификаты, ключи, коонтейнеры и т.п.)
    ├── certs       - директория для хранения сертификатов SSL
    └── keys        - директория для хранения ключей
        ├── public  - открытые ключи
        └── secret  - закрытые ключи (обязательно должны быть зашифрованы)
